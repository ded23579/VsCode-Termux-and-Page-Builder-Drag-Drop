# VsCode-Termux-and-Page-Builder-Drag-Drop

Sebuah proyek web sederhana yang menyediakan informasi tentang penggunaan Visual Studio Code di Termux dan teknik drag-and-drop untuk deployment.

## Deskripsi

Proyek ini berisi tiga halaman utama:
1. **Halaman Utama (index.html)** - Profil singkat tentang Dedi IR
2. **VS Code di Termux (vscode-termux.html)** - Panduan lengkap tentang penggunaan VS Code di lingkungan Termux
3. **Drag and Drop Deployment (drag-drop.html)** - Informasi tentang teknik deployment menggunakan metode drag and drop

## Fitur

- Tampilan responsif yang dapat diakses dari berbagai perangkat
- Slider otomatis pada halaman VS Code dan Drag & Drop
- Kode terminal yang dapat disalin dengan satu klik
- Iklan yang diperbarui otomatis setiap 10 detik
